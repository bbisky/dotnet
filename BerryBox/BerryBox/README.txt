BerryBox 黑莓手机工具箱

*******************************************************************************	
web: http://oteam.cn
Author:Sky
*******************************************************************************

Requirements
 1. Windows2000及以上 
 2  .Net Framework 2.0及以上版本
 
功能
 1. Cod安装工具
	* 安装本地cod文件
	* 删除手机已安装的软件
	* 保存手机已安装的cod	
	
 2. Jad/Alx生成工具
    * 根据指定目录的COD文件生成.alx文件,方便使用桌面管理器安装
	* 根据指定目录的cod文件生成jad文件，直接在手机上安装jad
	* cod文件会自动检测并拆解为最小单元
 
 3. 系统工具
	* WIPE手机
	* 无线电开关
	* 同步设备时间
	* 手机屏幕截图(可以设置保存格式)
	
 4. OTA软件下载工具
	* OTA下载软件功能, 支持设置机型和系统版本(对判断UserAgent参数的网站特别有用，如Google全系列的软件)
	
 5. USB接口驱动自动安装功能，完全脱离DM
 
 6. Jar2Cod Jar转换cod的工具*
	*该功能需要安装JRE以及Blackberry JDK中的rapc,net_rim_api.jar
